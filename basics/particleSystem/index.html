Particle System

A particle system is a technique to represent smoke, fire, rain, fireworks... or any phenomena that can be represented as a cloud of discrete small elements that combined appear as a single entity or follow some group dynamics.

The emitter is a location where the particles are created. Each particle can be rendered in multiple ways. Here we use quads that are always facing the camera also reffered as billboards.

Particles Texture

We can apply a texture to the particles to simulate different effects

Multiple Particle Systems

We can make effects composed of multiple particle systems. In this case we have a particle system for the shells and another one for the stars. One the shell particle finishes an event triggers the start of the stars

Particles Texture

We can apply a texture to the particles to simulate different effects
